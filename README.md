# Turntable
Turntable module driven by a stepper motor.

The top plates are interchangeable. I supply two variants 120/240mm.
.STEP file for custom top plate creation available!

Connected via a single custom ethernet cable to any microcontroller.
I use an ESP32 (M5Stack P4 with POE).

You can use my ESPHome interface (uses additional hardware rotary encoder) as a starting point for your own explorations:


I am planning to release a stand-alone device in the future (with integrated controls and ESP32) but this device is deliberately to connect to an external controller.



Hardware:

* Stepper Motor (JK42HS40-1004A-02F)
* Ethernet Keystone (deleyCON Tool-Free)
* TMC2209 Driver (TENSTAR Module V2.0)
* 170 Breadboard
* 3x M4 Flathead Screws (10-20mm)
* Cables, Wires
